# CorruptedUnzip

Build:

Open terminal or command prompt and run ...  
```
gradlew build
```

Usage:

You will find corruptedunzip-1.0-SNAPSHOT.jar in build/libs/  
Then run ... 
```
java -jar corruptedunzip-1.0-SNAPSHOT.jar [-o] [-q] zipfilename.zip [outputdir]

    Where:  -q => quiet
            -o => overwrite
```
